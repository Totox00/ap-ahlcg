# Arkham Horror The Card Game

## Where is the options page?

The [player options page for this game](../player-options) contains all the options you need to configure and export a config file.

## What does randomization do to this game?

Instead of playing campaigns linearly, each scenario is individually unlocked by items, as well as various locks within the scenarios.
Checks are taking clues from locations (locations with more clues have more checks), most sources of victory points, as well as some additional checks which are shown in the client when they are possible.
To goal a slot you must win a number of campaigns (usually requiring the final scenario unlock and some unlock items).

## What content is supported?

Currently only Night of the Zealot is supported.

## What Arkham Horror The Card Game items can appear in other players' worlds?

- Unlocks for scenarios
- Unlocks within campaigns
- Scenario cards within campaigns
- Experience points
- Additional Chaos Tokens
- Trauma
